# Aseprite test fixtures

Small, original `.aseprite` files that exercise the native importer's non-RGBA
paths (the 32-bpp happy path is covered elsewhere):

- `indexed_test.aseprite` -- 8-bpp indexed: a palette chunk plus transparent index 0
  (colored 4x4 blocks on a transparent border).
- `grayscale_test.aseprite` -- 16-bpp grayscale (value + alpha): a horizontal
  black-to-white gradient.
- `multiply_test.aseprite` -- 32-bpp RGBA with two layers, the top one in **Multiply**
  blend mode (left half darkened over an orange base, right half unchanged).

They are generated by [`scripts/gen_aseprite_fixtures.py`](../../../../scripts/gen_aseprite_fixtures.py)
(stdlib only; cels are ZLIB-compressed, so the inflate path runs too) and follow the
open `.ase` spec, so the Aseprite app can open them as well. The AssetProcessor builds
each to `diorama/examples/aseprite/<name>.dioramasheet` (+ `.streamingimage`); assign
the sheet to an Aseprite component or the image to a Sprite. These are original content
with no third-party IP. See [Docs/howto/19-aseprite.md](../../../../Docs/howto/19-aseprite.md).
